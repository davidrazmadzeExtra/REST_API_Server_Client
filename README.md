# REST_API_Server_Client

Example of REST API written in JS using Express.js.

Clients:

- Swift iOS App
- Objective-C iOS App
- Python
- JavaScript
- Ruby

## Running code

### Starting server

`npm install`
`node index.js`

### Clients
`python3 main.py`
`ruby main.rb`
`node client.js`

Run iOS Apps in Xcode
